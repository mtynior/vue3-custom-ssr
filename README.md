# Simple SSR Vue 3 app with routing and PWA

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies client for production
```
yarn build
```

### Compiles and minifies client and server for production
```
yarn build:all
```

### Run server
```
yarn start
```

### Run your unit tests
```
yarn test:unit
```

### Lints and fixes files
```
yarn lint
```
